# 实时流量监控系统用户指南

## 系统概述

实时流量监控系统是一个基于Web的网络流量监控平台，提供实时流量统计、安全监控、系统性能监控等功能。系统采用现代化的技术栈，支持一键部署，适合各种规模的网络环境。

## 功能特性

### 核心功能
- **实时流量监控**: 监控网络接口的实时流量数据
- **历史数据分析**: 查看历史流量趋势和统计
- **安全告警**: 检测异常流量和安全威胁
- **系统监控**: 监控CPU、内存、磁盘使用情况
- **多用户支持**: 支持多用户登录和权限管理

### 技术特性
- **响应式设计**: 支持桌面和移动设备访问
- **实时更新**: 数据自动刷新，无需手动刷新页面
- **安全认证**: JWT令牌认证，支持会话管理
- **数据可视化**: 丰富的图表和统计信息展示

## 快速开始

### 1. 访问系统

部署完成后，通过浏览器访问：
- **本地访问**: http://localhost
- **远程访问**: http://your-server-ip

### 2. 用户登录

系统提供两个默认账户：

**管理员账户**
- 用户名: `admin`
- 密码: `admin123`

**监控员账户**
- 用户名: `monitor`
- 密码: `monitor123`

> **安全提示**: 首次登录后请立即修改默认密码

### 3. 界面导航

登录成功后，您将看到主监控面板，包含以下区域：

- **顶部导航栏**: 显示系统标题和用户操作
- **系统概览**: CPU、内存、磁盘使用率统计
- **流量统计**: 发送/接收流量和带宽统计
- **实时图表**: 流量趋势图表
- **系统信息**: 详细的系统状态信息
- **安全告警**: 安全事件和告警信息

## 功能详解

### 流量监控

#### 实时流量查看
1. 在主面板的"实时流量图表"区域查看当前流量
2. 图表显示以下信息：
   - 发送带宽 (Mbps)
   - 接收带宽 (Mbps)
   - 发送包数
   - 接收包数

#### 接口选择
1. 在"选择网络接口"下拉菜单中选择要监控的接口
2. 选择"全部接口"查看所有接口的汇总数据
3. 选择特定接口查看该接口的详细数据

#### 自动刷新控制
1. 点击"开始自动刷新"按钮启用自动数据更新
2. 点击"停止自动刷新"按钮停止自动更新
3. 手动点击"刷新"按钮立即更新数据

### 系统监控

#### 系统资源监控
系统概览区域显示：
- **CPU使用率**: 当前CPU使用百分比
- **内存使用率**: 当前内存使用百分比
- **磁盘使用率**: 当前磁盘使用百分比
- **网络接口数**: 系统网络接口总数

#### 状态指示
- 🟢 **绿色**: 正常状态 (< 50%)
- 🟡 **黄色**: 警告状态 (50% - 80%)
- 🔴 **红色**: 危险状态 (> 80%)

### 安全监控

#### 安全告警
系统会自动检测以下安全事件：
- **高带宽告警**: 检测到异常高的网络带宽使用
- **流量激增**: 检测到流量突然增加
- **连接异常**: 检测到异常的连接模式
- **DDoS攻击**: 检测到可能的分布式拒绝服务攻击
- **端口扫描**: 检测到端口扫描活动
- **数据泄露**: 检测到可能的数据泄露活动

#### 告警级别
- 🔴 **Critical**: 严重告警，需要立即处理
- 🟠 **High**: 高级告警，需要关注
- 🟡 **Medium**: 中级告警，建议处理
- 🔵 **Low**: 低级告警，可忽略

#### 告警处理
1. 查看告警详情，了解告警原因
2. 根据告警类型采取相应措施
3. 对于误报，可以忽略或调整检测阈值

### 数据管理

#### 历史数据查看
1. 系统自动保存最近1小时的流量数据
2. 数据按时间序列存储，支持趋势分析
3. 数据自动清理，避免占用过多存储空间

#### 数据导出
- 当前版本暂不支持数据导出功能
- 未来版本将支持CSV、JSON等格式导出

## 高级配置

### 用户管理

#### 修改密码
1. 联系系统管理员修改用户密码
2. 或通过配置文件修改默认密码

#### 添加用户
在 `backend/main.py` 中添加新用户：
```python
USERS = {
    "admin": hashlib.sha256("admin123".encode()).hexdigest(),
    "monitor": hashlib.sha256("monitor123".encode()).hexdigest(),
    "newuser": hashlib.sha256("newpassword".encode()).hexdigest()
}
```

### 安全配置

#### 调整检测阈值
在 `backend/security_monitor.py` 中调整检测参数：
```python
# 高带宽检测阈值
if bandwidth_sent > 1000000000:  # 1GB/s

# 包速率检测阈值
if avg_rate > 10000:  # 每秒超过10000个包
```

#### 告警通知
- 当前版本暂不支持邮件或短信通知
- 未来版本将支持多种通知方式

### 性能优化

#### 数据收集频率
- 默认每5秒收集一次数据
- 可在前端界面调整刷新频率
- 高频率收集会增加系统负载

#### 历史数据保留
- 默认保留最近1小时的数据
- 可在配置文件中调整保留时间
- 建议根据系统资源情况调整

## 故障排除

### 常见问题

#### 1. 无法登录
**问题**: 输入正确用户名密码后仍无法登录
**解决方案**:
- 检查网络连接
- 确认服务器正在运行
- 检查浏览器控制台是否有错误信息

#### 2. 数据不更新
**问题**: 页面显示数据不更新
**解决方案**:
- 检查自动刷新是否启用
- 手动点击刷新按钮
- 检查后端服务是否正常运行

#### 3. 图表不显示
**问题**: 流量图表区域显示空白
**解决方案**:
- 检查浏览器是否支持JavaScript
- 清除浏览器缓存
- 检查网络连接

#### 4. 告警过多
**问题**: 系统产生大量告警
**解决方案**:
- 调整检测阈值
- 检查网络环境是否正常
- 联系管理员检查配置

### 性能问题

#### 1. 页面加载慢
**解决方案**:
- 检查网络带宽
- 关闭不必要的浏览器标签页
- 使用现代浏览器

#### 2. 数据延迟
**解决方案**:
- 检查服务器性能
- 调整数据收集频率
- 优化网络配置
